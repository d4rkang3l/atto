A small and fast subset-of-Scheme JIT interpreter

About

Atto is a small and fast subset-of-Scheme interpreter, leveraging the power of DynASM. It is primarily developed as a diploma project at the Faculty of Electronics, Telecommunications and Information Technology.

The language implements only a part of the R5RS, avoiding impure functions and features, and thus enabling lazy evaluation and memoization.

Try it out!

$ git clone https://github.com/deveah/atto.git
$ cd atto
$ make
$ ./atto examples/fib.atto
